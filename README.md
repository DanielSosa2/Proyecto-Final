## Sobre el proyecto

ALMA TREK es un sitio web sobre caminatas y circuitos naturales.  
El objetivo del proyecto fue poner en práctica todo lo aprendido en HTML, CSS/Sass, animaciones, responsive y SEO.  



## Qué incluye

- Hero con animación de texto  
- Sección de valores y servicios  
- Caminatas con cards interactivas  
- Galería de imágenes  
- FAQs con toggle  
- Contacto con formulario y mapa  
- Diseño responsive para tablet y móvil  
- Animaciones con Anime.js / AOS  



## Tecnologías

- HTML5 / CSS3 / Sass  
- Bootstrap 5  
- JavaScript para animaciones  
- Anime.js / AOS  
- Deploy en Vercel  



Notas
Se incluyó main.css (compilado desde Sass) y nativo.css como ejemplo de CSS plano.
